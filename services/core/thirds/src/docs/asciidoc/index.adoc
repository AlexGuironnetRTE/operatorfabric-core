Third service is used by third party service to customize some parts of OperatorFabric to fit their needs.
It allows to refine Operator Fabric behavior regarding:

* Published card details rendering
* Card available actions and rendering
* I18n resources for card rendering
* Additional menu entries
* Notification sound (not yet implemented)
